Release Notes – Version 1.0.0

Initial Release – Stock Tool Window

Features:

Display real-time stock prices, change percentages, and P/L information directly in the Visual Studio status bar.

Support customizable refresh intervals; information updates dynamically without switching windows.

Red for gains, green for losses, providing an intuitive view of market trends.

Monitor multiple stocks simultaneously.

Automatically clears status bar information when the tool window is closed, keeping the interface clean.

Notes:

Compatible with Visual Studio 2022 (Community / Professional / Enterprise).

Future versions will add more configurable features, such as custom colors and stock list management.
